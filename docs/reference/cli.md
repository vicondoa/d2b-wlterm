# CLI reference

Current skeleton commands:

- `d2b-wlterm name [seed]` prints a deterministic friendly name.
- `d2b-wlterm waybar` prints a minimal Waybar JSON payload.
- `d2b-wlterm open` exercises the already-attached Open model stub.
- `d2b-wlterm stop` exercises the Stop confirmation model stub.
- `d2b-wlterm config` prints the default config scaffold.

The CLI is intentionally small until the d2b public socket contract is wired via
`d2b-toolkit`.
